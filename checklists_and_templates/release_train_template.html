Release Train Process Overview

  • Day 0 - Scope Lock: Feature scope is finalized and locked for the release
  • Day 15 - Code Freeze: No new code changes, only bug fixes allowed
  • Day 20 - QA Freeze: QA testing complete, ready for stakeholder review
  • Day 25 - Stakeholder Demo: Final review and approval from stakeholders
  • Day 30 - Production: Release deployed to production environment
